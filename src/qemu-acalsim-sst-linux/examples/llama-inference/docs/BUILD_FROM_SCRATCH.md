# Build PyTorch from Scratch

See the complete PyTorch RISC-V build guide that was used to create this workspace.

This guide covers all the necessary patches and build steps for PyTorch 2.4 on RISC-V.

## Quick Reference

The image includes PyTorch 2.4 built with:
- Python 3.13.9
- RISC-V architecture support
- All necessary patches applied
- OpenBLAS for CPU math operations

If you want to rebuild or update PyTorch, you can find the complete build instructions
in the documentation that was used during the creation of this workspace.
